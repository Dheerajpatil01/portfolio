# Portfolio Website - Dheeraj Pravin Patil

A modern, responsive, and beautifully designed personal portfolio website built with React, Tailwind CSS, and Framer Motion. 

## 🚀 Features

- **Fully Responsive**: Mobile-first design that looks great on any screen.
- **Dark/Light Theme**: Built-in toggle for dark and light modes.
- **Smooth Animations**: Animated elements utilizing Framer Motion.
- **Clean Structure**: Component-based architecture for easy maintenance.
- **Modern UI**: Soft gradients, glassmorphism cards, and premium aesthetics.

## 🛠️ Required Technologies

- Node.js (v16+)
- npm (Node Package Manager)

## 📦 Setup Instructions

1. **Clone the repository** (if not already downloaded):
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. **Install all dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📁 Project Structure

- `src/components/` - Shared UI components (Navbar, Footer, ThemeToggle)
- `src/components/sections/` - Individual page sections (Hero, About, Projects, Skills, etc.)
- `src/index.css` - Global Tailwind utilities and base styles
- `public/resume.pdf` - Placeholder for your actual resume

## 📝 Customization

- **Resume**: Replace the `public/resume.pdf` placeholder with your actual PDF resume.
- **Theme Colors**: Open `tailwind.config.js` to tweak the primary, secondary, and hover colors.
- **Content**: Update texts directly inside the respective component files in `src/components/sections/`.
